# 🌟 Mindful Mini Dashboard

A beautiful, interactive web application that helps you practice mindfulness through task management, mood tracking, and gratitude exercises. All data is stored locally in your browser.

## ✨ Features

### 1. 📋 Task Manager
- Add and delete tasks
- Tasks persist in localStorage
- Clean, intuitive interface
- Clear all tasks option

### 2. 😊 Mood Tracker
- Log your current mood from 5 options
- View recent mood history with timestamps
- Get inspirational quotes (refresh anytime)
- Mood data persists across sessions

### 3. 🌟 Gratitude Score
- Increment gratitude counter with +1 button
- Receive dynamic affirmations based on your score
- Generate random positive affirmations
- Reset score when needed
- Progress-based encouraging messages

### 4. 📅 Live Date Display
- Shows current date with weekday
- Updates automatically

## 🎨 Design Features
- **Clean UI**: Modern glass-morphism design with soft gradients
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile
- **Smooth Animations**: Hover effects and button feedback
- **Custom Scrollbars**: Polished scrolling experience
- **Color-coded Cards**: Distinct visual hierarchy

## 🛠️ Technologies Used
- **HTML5**: Semantic structure
- **CSS3**: Modern styling with Grid, Flexbox, and animations
- **Vanilla JavaScript**: No frameworks, pure DOM manipulation
- **localStorage**: Persistent data storage

- Desktop: > 700px - Full grid layout

## 💾 Data Persistence
All user data is saved in browser's localStorage:
- Tasks list
- Mood history entries
- Gratitude score
